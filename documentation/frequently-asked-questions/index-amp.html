General Questions

Is this software server-based?
No. HTML Calendar Maker Pro is a stand-alone application, run on a single Windows 95, 98, Me, NT, 2000, or XP workstation, that generates web pages that can be manually transferred (i.e., uploaded or copied) to a server that resides on either an intranet or the Internet.

Can I import calendars from other programs?
Calendars may be imported if they are in comma-separated value (CSV) format and they must contain the event followed by the date (in mm/dd/yyyy format) of that event. Microsoft Outlook can export calendars in this CSV format. However, HTML Calendar Maker Pro cannot accept multiple events that occur on the same day. In other words, each date can only be listed once in each CSV file. See the User’s Guide for more information on .cal files, which implement this CSV style.

Can I use this program in conjunction with a web page editor?
Certainly. A good way to customize your calendars further is to use a web page editor like Dreamweaver. To modify your calendar in a web page editor, first save the calendar as a web page file and then open the file in the editor of your choice. HTML Calendar Maker Pro does not need to be running in order to use a separate web page editor.

What elements are saved when saving a calendar file?
Every event for the current month is saved in a calendar (.cal) file, which includes any links, bullets, and all web page formatting. Color schemes are not stored in calendar files but as global settings instead. These elements are applied to your calendars when they are generated as web pages.

How do I print my calendars?
Although HTML Calendar Maker Pro does not have a print feature, your Internet browser does. To print a calendar, generate it as a web page, preview it in your default browser, and print it from your browser.

When the web page is generated, why are the columns different widths?
Varying column widths may appear in your browser for many different reasons. This is caused by your Internet browser’s attempt to best-fit elements within the width of each column. This is usually caused by:

  • Very wide pictures
    (consider reducing size)
  • Long web page addresses
    (better implemented with word links of one or two words)
  • Long words
    (consider hyphenating)

    • How do I make a calendar for a month in the past or future?
    Although HTML Calendar Maker Pro displays the current month when first started, you can choose a month in any year in either the past or the future. To make a calendar for another month, click on the File menu and choose “New.”

    Registration Questions

    How can I transfer my registered copy to a new computer?
    All you need is your original registration code. Download the latest version of the program and install it on your new computer. When you run the newly-installed program, it will ask you for your registration code. Once the code is entered, you will have full use of HTML Calendar Maker Pro once again. Please be sure to discontinue use of the program on the old computer, since each registration code may only be used on one computer. Obviously, if you are discarding the old computer, discontinuing use will not be an issue. However, if you wish use HTML Calendar Maker Pro on more than one computer, we ask that you purchase a registration code for each additional computer.

    Can I use my registration code on more than one computer?
    As stated in the license agreement, which can be found in the User’s Guide and displayed during installation of the program, it is asked that you purchase a registration code for each computer on which you plan to run Web Page Calendar Maker. Visit the order page for ordering and discount information for multiple registration codes.

    When downloading a newer version, what steps should be taken?
    When updating your previous version of HTML Calendar Maker Pro with a new version, it is typically best to uninstall the previous version first, To uninstall, go to the Control Panel and choose Add/Remove Programs. When uninstalling the program, You may also be prompted to perform a “rollback” to previous system settings, which is not necessary. Your registration code is stored even when the program is uninstalled. Unless your calendars are stored in the same folder as the actual program, you do not need to make a backup of your calendars. Once the latest version has been downloaded, run the file to install HTML Calendar Maker Pro. Your settings, including your registration code, will reappear when the new version program is started.

    Troubleshooting

    Why won’t it accept my registration code?
    To resolve this issue, ensure that all dashes are included and that there are no spaces in the registration code. Note that the registration code is commonly confused with the product code of HTML Calendar Maker Pro (3060-1) as well as the order number. If the program still states that your registration code is invalid, request a replacement code.

    Why do I get run-time error 429?
    When starting HTML Calendar Maker Pro, you might receive the following message:

       Run-time error ‘429’:
    ActiveX component can’t create object

    This is most likely due to a file not installing correctly. To resolve this, take all of the following steps:

  • Uninstall HTML Calendar Maker Pro.
    In Windows, go to the Control Panel, choose Add/Remove Programs, and remove HTML Calendar Maker Pro.
  • When uninstalling, choose the Custom uninstall method.
    This is the first step you see when you begin to uninstall. After choosing Custom, click the Next button.
  • Select All files.
    To ensure every file is removed, click on the Select All Files button and click Next.
  • Select All system files.
    This is the most important: ensure all of these files have been selected by clicking on the Select All Files button again and then click Next.
  • Directories do not need to be removed.
    Since additional files may have been saved in these particular folders, it is best to simply click Next without selecting any directories.
  • Registry Keys do not need to be removed.
    Click Next without selecting any Registry Keys.
  • Registry Keys do not need to be removed, again.
    Click Next again.
  • Click Finish.
    This will perform the actual uninstallation.
  • Restart the computer.
    In case any of the files were in use, the computer must be restarted in order to remove the in-use files.
  • Download and install.
    Download the latest version of HTML Calendar Maker Pro and install it.
