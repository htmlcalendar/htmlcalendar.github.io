Order Color Scheme Add-On Packs

Adding dramatic colors to your monthly web page calendars just got a little easier.

With each of the two Color Scheme Add-On Packs, you can add an additional 35 color schemes to your copy of HTML Calendar Maker Pro, which originally comes with 15.

These 70 new color schemes let you easily choose an entire set of colors for your calendars, altering the flavor with only the click of a mouse. Match your calendar with the style of your site or try something completely different from month to month! Feel free to try various styles when you please; your fonts and alignments will remain untouched as you experiment with these diversely varied color schemes.

With 35 color schemes in each of the two Add-On Packs, you will have a good number of unique styles for your future calendars.

Purchasing

After each Add-On Pack is purchased, you will be able to download it from a link that will be provided to you.

Installing

If you have not yet installed HTML Calendar Maker Pro, please install and run it before continuing with the installation of these Add-On Packs. When you have downloaded each pack, simply run it and follow the directions onscreen. If you ordered both Add-On Packs or have one that was previously installed, it will not matter which needs to be installed first. Upon successful completion of the installation, click the OK button and run HTML Calendar Maker Pro. You will find your new color schemes alongside the original schemes. We highly recommend that you save a copy of each of the Add-On Pack programs to disk for backup purposes. You will most likely need to run it again if HTML Calendar Maker Pro is reinstalled or updated later.
